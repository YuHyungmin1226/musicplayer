# 🎵 Python Music Player

Tkinter와 Pygame을 활용하여 제작된 심플하고 직관적인 뮤직 플레이어입니다.

## 주요 기능

- **재생 컨트롤**: 재생(▶), 일시정지(⏸), 정지(⏹), 이전 곡(⏮), 다음 곡(⏭)
- **플레이리스트 관리**: 
  - 개별 오디오 파일 추가 가능
  - 폴더 전체 추가 가능 (하위 폴더 포함 탐색)
- **재생 모드**:
  - **셔플(Shuffle)**: 무작위 재생 모드
  - **반복(Repeat)**: 한 곡 반복, 전체 반복, 반복 안 함 모드 지원
- **사용자 편의 기능**:
  - 볼륨 조절 슬라이더
  - 진행 상태 바 (클릭 시 원하는 위치로 이동 가능)
  - 현재 재생 시간 및 전체 곡 길이 표시

## 설치 방법

이 프로젝트는 Python 3.11 환경에서 최적화되어 있습니다.

1. **저장소 복제 또는 다운로드**
2. **가상 환경 설정 및 패키지 설치** (이미 설정된 경우 생략 가능):
   ```bash
   python3.11 -m venv .venv
   source .venv/bin/activate  # macOS/Linux
   pip install -r requirements.txt
   ```

## 실행 방법

가상 환경이 활성화된 상태에서 아래 명령어를 입력하세요:

```bash
python music_player.py
```

## 시스템 요구 사항

- **Python**: 3.11 이상 권장
- **라이브러리**: 
  - `pygame`: 오디오 재생 엔진
  - `tkinter`: GUI 라이브러리 (기본 설치 포함)

## 참고 사항

- 지원하는 파일 형식: `.mp3`, `.ogg`, `.wav`, `.flac`
- IDE(VS Code 등)에서 실행 시 인터프리터를 프로젝트 폴더 내의 `.venv`로 설정해야 `pygame` 라이브러리를 정상적으로 인식합니다.
